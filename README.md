# Simulador de Investimento

Aplicativo Flutter para simular investimentos mensais, permitindo visualizar o montante acumulado com e sem juros ao longo do tempo.

---

## 📌 Contexto

Uma alternativa ao financiamento é poupar e investir o dinheiro ao longo do tempo. Para bens que não são essenciais, é possível planejar a compra à vista acumulando capital.

---

## 🎯 Objetivo

Desenvolver um aplicativo que receba:

* Valor mensal investido
* Número de meses
* Taxa de juros mensal

E exiba:

* Montante acumulado sem juros
* Montante acumulado com juros compostos

---

## 🚀 Tecnologias

* Flutter
* VS Code
* Android Studio

---

## 🧩 Recursos Utilizados

* Interface com Widgets Flutter
* Campos de entrada (`TextField`)
* Botões (`ElevatedButton`)
* Cálculos matemáticos
* Atualização de estado (`setState`)

---

## ▶️ Como executar

1. Clone o repositório
2. Abra no VS Code
3. Abra o terminal (Ctrl + ")
4. Execute:

flutter pub get
flutter run

5. Escolha um emulador ou navegador
